# Blog 프로젝트

## 기술스택
- JDK 11
- Springboot 2.7.8
- MyBatis
- 테스트 h2 디비
- 프로덕션 MySQL 디비
- JSP

## 모델링
### 1단계
- User
- Board
### 2단계
- Reply
### 3단계
- Love

## 기능정리
### 1단계 (완료)
- 회원가입, 로그인, 글쓰기, 글목록보기, 글상세보기, 글삭제, 글수정
### 2단계 (시작)
- 댓글쓰기, 댓글목록보기, 댓글삭제
### 3단계
- 좋아요하기, 좋아요취소, 좋아요보기
### 4단계
- 아이디중복체크, 회원수정하기, 페이징
### 5단계
- 검색
